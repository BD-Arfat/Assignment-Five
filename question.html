What is dom in javascript?

The DOM (Document Object Model) in JavaScript is a programming interface for web documents, allowing you to interact with and manipulate the structure and content of a webpage.

how do you select an element from the dom

To select an element from the DOM in JavaScript, there are several methods you can use depending on what you're trying to select. Here are the most common methods: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll()

what is even delegation in the context of the Dom, and why is it usefull

Event delegation is a technique where you handle events (like clicks) on many elements (like buttons) by placing just one event listener on their parent. This makes it easier and faster, especially when you have many elements.
